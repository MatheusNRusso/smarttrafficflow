What it does

Three lenses on the same city.

From a single bus moving stop-by-stop to a city-wide congestion view, SmartTrafficFlow surfaces the same dataset at every scale.

01 / animation

Stop-by-stop animation

Each bus follows its real route geometry, easing between stops so you can watch a line breathe across the network in real time.

GTFS-RT · interpolated · 60 fps
02 / planner

A → B route planner

Drop two pins anywhere in the metro. We resolve the bus lines that connect them, ranked by travel time, transfers and live load.

graph search · multi-modal · < 200 ms
03 / insights

Traffic insights

Heatmaps and corridor-level metrics expose where the network is congested, where buses bunch, and where headways break down.

live · congestion index · per-corridor

Open data

Built on what the city already publishes.

SmartTrafficFlow consumes the public GTFS and live AVL feeds released by the Prefeitura do Rio de Janeiro — no scraping, no proprietary dataset. Reproducible by design.
